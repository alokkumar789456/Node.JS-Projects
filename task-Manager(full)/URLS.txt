URL: POST http://localhost:5000/api/auth/signup
{
  "name": "John Doe",
  "email": "johndoe@example.com",
  "phone": "1234567890",
  "password": "mypassword"
}


URL: POST http://localhost:5000/api/auth/verify-otp
{
  "email": "johndoe@example.com",
  "otp": "123456"  // Replace with actual OTP
}

URL: POST http://localhost:5000/api/auth/login
{
  "email": "johndoe@example.com",
  "password": "mypassword"
}

{

}

URL: POST http://localhost:5000/api/books
headers:
{
  "Authorization": "Bearer <admin-jwt-token>"
}
Body (Form-Data for Multer):

Key: bookName (String) - optional
Key: genre (String) - optional
Key: authorName (String) - optional
Key: ISBN (String) - optional
Key: rate (Number) - optional
Key: image (File) - optional
